These icons are a curated collection from Font Awesome.
How to add more icons.

It is also possible to use svg icons from outside gwds library. Just provide the path to the svg, by using the "src" attribute, and not providing a "name". To use the intrinsic icon color also add the "autocolor" attribute.

How to add more icons to the library:

  1. download the icon from font awesome in svg format.
  2. clone gwds repo to your machine.
  3. copy the new svg icon to "src/assets/icons".
  4. open the terminal, and do "npm run build:rollup". This will build the project with the updated icons folder.
  5. push the new changes to the repo.
